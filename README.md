# ML Model Monitoring with Evidently AI

An end-to-end ML monitoring project using Evidently AI.

## Reports Generated
- Data Drift Report
- Data Summary Report
- Classification Report
- Regression Report
- LLM Text Evaluation Report

## Tech Stack
Python 3.12, Evidently AI 0.7.21, Scikit-learn, Pandas, Plotly
