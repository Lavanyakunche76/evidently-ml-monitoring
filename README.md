# ML Model Monitoring with Evidently AI

An end-to-end ML monitoring project using Evidently AI.

## Reports Generated
- Data Drift Report - detects feature distribution changes
- Data Summary Report - data quality and statistics
- Classification Report - accuracy, precision, recall, F1-score
- Regression Report - MAE, RMSE, error distribution
- LLM Text Evaluation Report - sentiment, text length, toxicity

## Tech Stack
- Python 3.12
- Evidently AI 0.7.21
- Scikit-learn
- Pandas
- Plotly
